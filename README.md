# CityPulse AI

AI-powered news categorization and route optimization system for journalists.

## Project Structure

- `Q2_News_Classifier.ipynb` - Naive Bayes models for news categorization
- `Q3_Pathfinding.ipynb` - A* and UCS pathfinding algorithms
- `Q4_System_Integration.ipynb` - Complete system with GA for TSP
- `CityMap.ipynb` - City distance data and heuristics
- `News_Categorizer_RDF.xml` - Training data
- `Daily_News.csv` - Test data

## Features

- Multi-model news classification
- Optimal pathfinding within cities
- TSP optimization for multi-city routes
- Interactive user interface

## Technologies

- Python 3.12
- scikit-learn for ML
- rdflib for RDF parsing
- Custom search algorithms
- Genetic algorithms for optimization
